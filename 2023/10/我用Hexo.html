如何删除 GitHub 上的历史提交记录

在《黑猫警长》里，一只耳为了逃避追捕，先踏出几步脚印，眼珠子一转，又折返回来，走另一条路，边走边用尾巴扫去自己的新脚印。

在 GitHub 上删除所有 commit ，基本思路也是一样。

首先要新建一条孤儿分支，并设置为默认分支，然后删除充满历史包袱的旧分支，最后将默认分支改成旧分支的名字。
